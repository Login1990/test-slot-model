# Small slot model

## Prerequisites

1. Node >= 24.7.0 (other version not tested)
2. npm >= 11.5.1

## Install and run 

1. Open terminal in directory and type:
```node
npm install
```
2. To spin manually:
```node
node casino.js
```
Press `ENTER` to spin, press `q + ENTER` to quit if you don't like `CTRL + C` or `CMD + C`
3. To mass spin:
```node
node simulate_rtp.js
```
